# BASH SHORTCUTS
| COMMAND          | FUNCTION                |
|------------------|-------------------------|
| ctrl + d         | close the bash shell    |
| ctrl + alt + t   | create a new bash shell |
| ctrl + shift + t | 打开新的标签页          |
| ctrl + s         | 暂停屏幕输出            |
| ctrl + q         | 继续屏幕输出            |
